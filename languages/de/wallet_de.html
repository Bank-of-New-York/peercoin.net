Installation


Windows

Windows

  • Öffne oder extrahierePeercoin_v0.7.1_win.zip
  • 32 bit oder 64 bit Version installieren
  • Standard-Installationspfad istC:\Program Files\PPCoin oderC:\Program Files\Peercoin
  • Starten Siepeercoin-qtoderPeercoinvom Startmenü oder aus dem Installationsverzeichnis

Daemon

  • Erstellen Sie ppcoin.conf in%APPDATA%\PPCoin
  • Starten Sie daemon\peercoind
  • Ihr Wallet befndet sich in C:\Users\<username>\AppData (verstecktes Verzeichnis).
  • Öffnen Sie das Verzeichnis mit Startmenü -> Ausführen... (oder drücken Sie Windowstaste+R), tippen Sie%APPDATA%\PPCoinin das Feld und drücken Sie Enter.

Weitere Hilfe können Sie finden in der Peercoin Dokumentationoder indem Sie in unserem Forumfragen.
OS X

OS X

  • Öffnen oder extrahieren Sie peercoin-v0.7.1ppc-osx-gitian.zip
  • Legen Sie Peercoin-Qt.dmg an, indem Sie es öffnen
  • Ziehen Sie Peercoin-Qt.app oderPeercoin.app zu Anwendungen
  • Ziehen Sie Peercoin-Qt.app oderPeercoin.app zu Anwendungen
  • Gehen Sie zu Anwendungen und öffnen Sie Peercoin-Qt.app oderPeercoin.app

Daemon

  • Entpacken Siepeercoind.zip
  • Erstellen Sie ppcoin.conf in ~/Library/Application Support/PPCoin
  • Starte peercoind
Ihr Wallet befindet sich in ~/Library/Application Support/PPCoin. Öffnen Sie das Verzeichnis, drücken Sie STRG+Shift+G im Suchfeld, fügen Sie den Pfad ein, und drücken Sie Enter.

Weitere Hilfe können Sie finden in der PeercoinDokumentation oder indem Sie in unserem Forumfragen.
Linux

Linux, allgemein

  • Entpacken Sie peercoin-v0.7.1ppc-linux-gitian.zip
  • Starten Sie bin/32/peercoin-qt (benötigt libqt4-gui)

Daemon

  • Erstellen Sie peercoin.conf in ~/.peercoin
  • Starten Sie bin/32/peercoind
Ihr Wallet befindet sich in ~/.peercoin.
Anmerkung: 64 bit Startdateien befinden sich in bin/64/{peercoin-qt,peercoind}.

ArchLinux

yaourt -S peercoin-qt

Weitere Hilfe finden Sie in der Peercoin Dokumentation oder indem Sie in unserem Forum nachfragen.
